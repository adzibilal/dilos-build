Chip Components

DilosUI comes with a chip component that can be used to trigger actions.

Default​

    <div class="flex items-center gap-5">
        <div class="d-chip xs gray"> 
            Label
        </div>

        <div class="d-chip md gray"> 
            Label
        </div>

        <div class="d-chip lg gray"> 
            Label
        </div>
    </div>
    <div class="flex items-center gap-5">
        <div class="d-chip xs primary"> 
            Label
        </div>

        <div class="d-chip md primary"> 
            Label
        </div>

        <div class="d-chip lg primary"> 
            Label
        </div>
    </div>
